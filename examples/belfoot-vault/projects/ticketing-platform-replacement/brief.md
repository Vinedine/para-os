# Ticketing platform replacement

Replace the legacy ticketing platform (in production since 2014) with a modern, API-first replacement that supports dynamic pricing, secondary market, and integration with the Salesforce CRM and the new cashless wallet.

## Why now

- **End of support.** The current vendor announced EOL for our version in Q1 2027.
- **Integration debt.** Every other system (CRM, web store, matchday ops, cashless) has had to build awkward adapters. A modern API removes those.
- **Dynamic pricing** is a board priority for 2026-27. Current platform doesn't support it.
- **Secondary market.** Re-selling is currently a manual workaround. New platform handles it natively.

## Scope

In:

- Selection, contracting, and migration to a new platform.
- Integration with Salesforce CRM, the new cashless wallet, matchday operations.
- Data migration of season-ticket holder records (~24k accounts).
- Phased cutover: corporate hospitality first, then general admission.

Out:

- Replacement of the loyalty programme (separate workstream).
- The mobile app rebuild (separate workstream, dependent on this).
