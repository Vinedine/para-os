# Examples

Synthetic worked vaults. Every person, company, property, and number in here is fictional; the structure and conventions are the real thing.

- [`belfoot-vault/`](belfoot-vault/): a default (plain editable markdown) vault for a fictional consulting engagement: BelFoot FC, a Belgian football club running an IT modernisation programme. Two projects with briefs and actions, four stakeholder contact files, dated meeting records, and a vault `CLAUDE.md`. Open a Claude Code session in its root and run `/daily-brief` or `/triage` to see the skills work.

A **readonly-ipad** example vault is planned.
